Help Articles\Category - In-Game Issues\VeinMiner Doesn't Work on Certain Blocks 
Step 1: Type in the Command: /veinminer blocklist [Tool to be Used] add [ModName:ItemName] [Metadata (If Any)]*# 

* - The Command will Add that Particular Block to that Particular Tool's Blocklist. In Order to Get the Block's ModName:ItemName Data, get that Particular Block's Data Value, and do /give @p [Data Value] 1 [Metadata (If Any)].
# - Example - Lapis Caelestis Callainus. It's ModName:ItemName Data is ExtraUtilities:greenscreen, and Since the Lapis Caelestis Callainus has a Metadata of 9, the Command would be /veinminer blocklist pickaxe add ExtraUtilities:greenscreen 9.
Help Article Origin: KingWealthyturtle

Step 1: Go to "Settings..."*
Step 2: Click on "Snooper Settings..."*
Step 3: Click on "Allow Snooper: ON" to Disable Snooper (Which Will Stop the Console Spams)*

* - In the Minecraft Client, and not the Curse Client.

Help Articles\Category - Other Issues\Disabling Mods (NOT RECOMMENDED)
Step 1: Go to Curse\Minecraft\Instances\Universal Interdimensional Escape\mods
Step 2: Find the Mod .jar File that you Want to Disable
Step 3: Replace the Mod File's .jar Extension with a .disabled Extension*

* - Warning: Disabling Certain Mods May Cause Problems in the Modpack, such as Quests and Crafting Recipes being Glitched Out, as well as Causing your Save File to be Corrupted.
